#  Assignment Title

Design a dynamic web app like fashion store / E-Commerce product using HTML, CSS & JS.

---

#  Aim

To design and develop a dynamic e-commerce website using HTML, CSS, and JavaScript that allows users to browse products and interact with a shopping cart system.

---

#  Objectives

* To design a visually appealing user interface using HTML and CSS
* To implement dynamic functionality using JavaScript
* To display products dynamically using JavaScript arrays
* To implement Add to Cart functionality
* To update cart data in real-time

---

#  Technologies Used

* HTML
* CSS
* JavaScript

---

#  Description

This project is a simple e-commerce fashion store that displays products in a card layout. Each product includes an image, name, and price.

Users can add items to the cart, and the cart updates dynamically without reloading the page. The number of items in the cart and the cart contents are displayed in real-time using JavaScript.

The UI is designed using modern styling techniques such as gradients, shadows, and hover effects to enhance user experience.

---

#  Working

1. Products are stored in a JavaScript array
2. Products are dynamically displayed on the webpage
3. User clicks "Add to Cart"
4. Item is added to cart array
5. Cart count updates instantly
6. Cart items are displayed in cart section

---

#  Conclusion

The project successfully demonstrates a dynamic e-commerce interface using JavaScript. It provides a basic understanding of how real-world shopping platforms work and helps in learning DOM manipulation and event handling.
