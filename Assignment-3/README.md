# Fashion E-Commerce Web App

## Overview

This project is a dynamic fashion e-commerce web application designed to simulate a real online shopping experience. It provides a modern UI with multiple product categories, interactive features, and smooth navigation similar to popular fashion platforms.

---

## Features

* Homepage with Hero Section

  * Image slider with promotional banners
  * Category navigation (Men, Women, Footwear, etc.)

* Shopping Functionality

  * Add to cart with dynamic quantity update
  * Cart badge updates in real-time
  * Checkout section with total calculation

* Wishlist Feature

  * Add/remove products from wishlist
  * Data stored using local storage

* Search & Filters

  * Search products by name/category
  * Filter by size and price range
  * Sort products (price, popularity, discount)

* Product Details Page

  * Image preview with zoom
  * Size and color selection
  * Quantity selection

* Responsive Design

  * Works on desktop and mobile screens
  * Hamburger menu for smaller devices

---

## Concepts Used
