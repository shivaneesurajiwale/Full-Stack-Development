# 🚨 Women Safety Data Visualization Dashboard

## Overview

This project is an interactive data visualization dashboard designed for a real-life application focused on women safety. It presents safety-related data using charts and graphs to help understand patterns of incidents such as harassment, theft, unsafe areas, and emergencies.

The dashboard is built using Chart.js along with HTML, CSS, and JavaScript, providing a simple and responsive user interface.

---

## Problem Statement

Create an interactive data visualization dashboard for a real-life application like a weather dashboard. Visualize data using charts and graphs using libraries such as Chart.js or D3.js.

---

## Features

* Interactive Dashboard Interface

  * Clean and responsive layout
  * Dark and Light mode toggle

* Data Visualization using Chart.js

  * Pie Chart for alert type distribution
  * Line Chart for alerts over time
  * Bar Chart for area safety scores

* User Interaction

  * Add new alert manually
  * Simulate random alerts
  * Real-time chart updates

---

## Technologies Used

## Project Structure

* index.html – Main dashboard structure

## Working

* The dashboard initializes with predefined alert data
* Pie chart shows distribution of different alert types
* Line chart dynamically updates when alerts are added
* Bar chart represents safety scores of different areas
* Users can add alerts or simulate alerts
* Charts update in real-time using Chart.js
