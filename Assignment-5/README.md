#  Assignment Title

Create a simple functional college website / travel agency website / insurance website using Node.js and Express using any NoSQL data set like MongoDB if required.

---

#  Aim

To design and develop a personal portfolio website integrated with a backend system using Node.js, Express.js, and MongoDB to store and manage user data dynamically.

---

#  Objectives

* To design a responsive portfolio website using HTML and CSS
* To implement interactivity using JavaScript
* To integrate frontend with backend using Fetch API
* To develop a backend server using Node.js and Express.js
* To store and manage user data using MongoDB
* To understand full-stack web development concepts

---

#  Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB
* API Handling: Fetch API

---

#  Description

This project is a personal portfolio website that showcases education, skills, experience, and projects.

The website includes a functional contact form connected to a backend server built using Node.js and Express.js. When a user submits the form, the data is sent to the backend using the Fetch API.

The backend processes the request and stores the user data (name, email, and message) in a MongoDB database. This demonstrates full-stack integration including frontend, server-side logic, and database management.

---

#  Working

1. User opens the portfolio website
2. Navigates to the Contact section
3. Enters name, email, and message
4. Clicks "Send Message"
5. Data is sent to backend using Fetch API (POST request)
6. Express server receives and processes the data
7. Data is stored in MongoDB database
8. Success message is displayed to the user

---

#  Database (MongoDB)

* Database stores user contact details
* Fields stored:

  * Name
  * Email
  * Message
* Data is stored in collection (e.g., `contacts`)

---

#  Conclusion

The project successfully demonstrates full-stack web development by integrating frontend, backend, and database. It provides hands-on experience with Node.js, Express.js, and MongoDB, and shows how real-world web applications handle and store user data.
